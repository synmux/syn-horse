# syn.horse — mobile UI kit

iPhone-framed mockup of the syn.horse mobile app. 4 screens cycle via the bottom dots:

1. **feed** — your void, list view
2. **synth detail** — full-screen synth read view
3. **compose** — make a new synth (keyboard up)
4. **profile** — handle, bio, danger zone

uses the bundled iOS frame component. dark only — syn.horse refuses light mode.
