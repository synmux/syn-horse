# syn.horse — personal site UI kit

a personal site built on syn.horse aesthetics. the blog is **one section among many**, not the front door.

5 screens (top nav cycles):

1. **home** — name, current status, what i do, scattered things
2. **now** — what i'm working on right now (the /now page)
3. **projects** — grid of things i've made
4. **blog** — index of posts (a section, not a homepage)
5. **post** — a single blog post

voice stays in brand: lowercase, occasionally feral, allergic to thought-leadership copy.
